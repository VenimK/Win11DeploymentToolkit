Windows 11 Deployment Toolkit 
============================= 
 
This toolkit provides a set of scripts and tools to help deploy Windows 11 in various environments.

IMPORTANT: Before using this toolkit, you must place a Windows 11 ISO file in the "Windows11" folder.
If the folder doesn't exist, create it in the root directory of the toolkit.

See the Documentation.html file in the Docs folder for detailed information on how to use this toolkit.

New Features:
- Drive letter independence: The toolkit now works from any drive letter
- Progress indicators for long-running operations
- Enhanced documentation with troubleshooting steps
- Update checker to ensure you're using the latest version

To check for updates to the toolkit, run the CheckForUpdates.bat file.

To get started, run MainMenu.bat 
 
For detailed documentation, see Docs\Documentation.html
