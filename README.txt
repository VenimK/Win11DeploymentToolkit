Windows 11 Deployment Toolkit 
============================= 
 
This toolkit provides tools for Windows 11 deployment: 
- Windows 11 upgrade (preserves all data) 
- System compatibility check 
- Update extraction and installation 
- Custom answer file creation 
 
IMPORTANT: Before using this toolkit, you must place a Windows 11 ISO file in the "Windows11" folder.
If the folder doesn't exist, create it in the root directory of the toolkit.
 
To get started, run MainMenu.bat 
 
For detailed documentation, see Docs\Documentation.html
